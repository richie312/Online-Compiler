Missing Elements

Given an array C of size N-1 and given that there are numbers from 1 to N with one element missing, the missing number is to be found.

Input:
Define a function which takes two argument; array or list of numbers and Nth element.

Output:
Return the list of missing number/s.

Example:

Input:

array = [1,2,3,5,4] and N = 10

Output

[6, 7, 8, 9, 10]
