Missing Elements

Given an array C of size N-1 and given that there are numbers from 1 to N with one element missing, the missing number is to be found.
Argument:
Define a function missingNumbers(list, num) which takes two argument; array or list of numbers and Nth element. Return the list of missing number/s.

Example:

  • Input: [1,2,3,5,4], 10
  • Output: [6, 7, 8, 9, 10]
